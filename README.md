# GestureFlow: 基于深度学习的手势交互系统

这是一个为人工智能基础课程开发的实时手势识别应用，采用 React 前端与 MediaPipe 算法引擎。
## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
